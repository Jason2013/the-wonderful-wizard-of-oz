《绿野仙踪》阅读
================

《绿野仙踪》系列童话故事（又名《OZ国经典童话》），是美国作家莱曼•弗兰克•鲍姆(L. Frank Baum)在近二十年间陆续创作发表的奇幻冒险童话故事集，共十四本。《绿野仙踪》是这个系列的第一本，也是流传最广、最受欢迎的一本。
